# System Overview

The KALFA platform is a **multi-tenant SaaS event management system** built using:

- Laravel 12
- Livewire 4
- Tailwind CSS v4
- MySQL / PostgreSQL
- Twilio integrations
- Payment gateway (SUMIT)

### Main capabilities

- RSVP management
- Seat assignment
- Event tables
- Multi-organization support
- Billing and subscriptions
- Messaging integrations
